Estrutura Básica do HTML

(voltar)

O documento HTML se inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre que quiser codar em HTML, você sempre começará dessa forma e por esse código. Veja abaixo como a estrutura se inicia:

  1. <!DOCTYPE html>
  2. <html lang="pt-br">
  3.   <head>
  4.     <title>Título da página</title>
  5.     <meta charset="utf-8">
  6.   </head>
  7.   <body>
  8.     Aqui vai o código HTML que fará seu site aparecer.
  9.   </body>
  10. </html>

O HEAD representa o cabeçalho de um documento. Diferentemente do conteúdo exibido no navegador, o conteúdo dentro dessa tag não aparece na página quando ela é carregada. Em vez disso, o cabeçalho contém metadados sobre o documento.

O BODY representa o corpo de um documento. Diferentemente do conteúdo exibido no navegador, o conteúdo dentro dessa tag é o que aparece na página quando ela é carregada.

Falando sobre Tags

(voltar)

As tags geralmente vêm em pares, com uma tag de abertura "< tag >" e uma tag de fechamento "< / tag >". Algumas tags são auto-fecháveis e têm a forma "< tag />". Existem tags para diferentes propósitos, como < strong > para deixar um texto em negrito, < i > para itálico, < input > para abrir um campo de texto para o usuário, < img > para inserir imagens na página.

Atributos das Tags

(voltar)

As tags podem ter atributos que fornecem informações adicionais ou modificam seu comportamento. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação). Os Atributos possuem nome e um valor, existem Atributos que você vai usar praticamente sempre e existem outros que serão mais raros.

Se você utilizar o Visual Studio Code, ao abrir uma tag, a própria IDE já te sugere alguns atributos que podem ser inseridos naquela tag, a fim de facilitar o desenvolvimento do código.

Links

(voltar)

Os links em HTML são criados usando a tag "< a >", que é a abreviação de “âncora”. Eles são fundamentais para a navegação na web, permitindo que os usuários cliquem e sejam redirecionados para outra página, documento ou recurso.

  • Sintaxe Básica: Para criar um link, você usa a tag < a > com o atributo href, que especifica o URL do destino.
  • Âncoras: Você também pode criar links para seções específicas dentro da mesma página usando âncoras, basta preencher o href com "#idDoElementoAqui".
